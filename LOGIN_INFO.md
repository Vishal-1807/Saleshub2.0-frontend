## Features by Role

### Campaign Manager Features
- Create and manage campaigns with custom form builders
- Assign field agents to campaigns
- View all leads across all campaigns
- Push leads to external CRM systems
- Generate performance reports and analytics
- View agent efficiency metrics
- Monitor feedback trends

### Field Agent Features
- View assigned campaigns and leads
- Record customer visit outcomes (interested, not interested, not home)
- Add detailed feedback notes
- Track personal performance metrics

### Call Center Agent Features
- Access routed leads (interested/contacted status)
- Perform follow-up calls
- Update lead status (completed, in progress, lost)
- Track call center performance

## Important Note

**This application now uses 100% mock data** - no database connection required!
- All data is hardcoded in the application
- Changes made during the session are stored in browser memory only
- No data is persisted to any backend or database
- Perfect for demonstrations and testing role-based workflows
